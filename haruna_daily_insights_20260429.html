"바디로션 바르는 게 귀찮아서 안 바름" — 사용 행동 자체의 진입 장벽

  • "귀찮아서 안 바름"
  • "온몸에 미끌미끌 개싫음"
  • "끈적이기는 걸 싫어해서 여름에는 바디로션을 거의 바르지 않았"
  • "바르고 나면 없으면 안 되는 존재" (습관 형성 후 만족 대비군)
🌟 일상생활 컨텍스트: "샤워 직후 5분이 결정적 — 그 시간에 미끌거림이 손에 남으면 다음번엔 안 바르게 됨". 남성 사용자의 "온몸에 미끌미끌 개싫음" 표현이 비사용층의 본능적 거부감을 상징

  • 바디로션 안 바르는 남자들 (에펨코리아) — "귀찮아서 안 바름", "미끌미끌 개싫음" 비사용층 직접 인용 출처
  • 끈적이지 않고 촉촉한 지성 바디로션 (언파) — "끈적이는 걸 싫어해서 여름엔 거의 바르지 않았" 사용 포기 직접 출처
  • 바디로션, 크림 후기 긴글 (더쿠) — 끈적임으로 인한 사용 포기/회피 패턴 종합 출처
시장이 푸는 방식: 일리윤 세라마이드(가벼운 제형 무향), 피지오겔 DMT(끈적임 없음 강조), 부케가르니(가벼운 발림성 + 향)
해결되지 않은 갭: "바르는 행동 자체를 즐겁게" 만든 제품이 없음. 모든 광고가 "바르고 나서의 결과"만 강조 — 바르는 5초의 감각 경험을 콘텐츠로 풀어낸 사례 부재
하루나 진입 포인트 흰목이버섯 다당류의 "흡수 직후 손에 남지 않는" 텍스처 + "5초 만에 끝나는 바디케어" 콘텐츠로 비사용층 직접 공략. "귀찮아서 안 바르셨죠? 5초면 끝납니다" 카피

"끈적임 + 트러블 + 향 — 동시에 다 만족시킬 수 없는 3중고"

  • "세타필은 보습은 최고였는데 끈적거리고 트러블 유발"

  • "피지오겔 DMT는 끈적임 없고, 보습감이 좋아" (모범 사례)
🌟 일상생활 컨텍스트: "다시 샤워해야 할 것 같은" — 바른 직후 향이 너무 강하거나 끈적이면 소비자는 즉시 "씻고 싶다"는 본능 반응을 보임. 향이 단순 호불호가 아니라 사용 지속성을 좌우
📍 직접 확인 가능 리뷰
  • 바디로션 크림 후기 긴글 — 6개 제품 비교 (더쿠) — "다시 샤워해야 할 것 같은", "허벅지 찢어지는 줄" 인용 출처
  • 끈적임 없는 고보습 바디로션 추천 (언파) — "끈적임과 텁텁함 때문에 챙겨 바르기가 쉽지 않아요" 출처
  • 여름에 끈적이지 않는 바디로션 추천 (언파) — "여름엔 답답함, 가슴 부위에 트러블도 올라옴" 출처
시장이 푸는 방식: 피지오겔 DMT(끈적임 0 + 무향), 일리윤 세라마이드(저자극 + 가벼움), 블랑카우 모이스처 리페어(흡수력 강조)
해결되지 않은 갭: "끈적임 0 + 트러블 0 + 향까지 만족스러운" 제품은 시장에 거의 없음. 무향(피지오겔)은 향 만족 ❌, 향 좋은 제품(러쉬)은 끈적임/트러블 ❌
하루나 진입 포인트 식자재 기반 자연향 + 흰목이버섯 다당류로 3중고를 동시에 해결한 포지션. "샤워 후 바로 옷 입어도 OK + 트러블 없는 + 자연스러운 향"이라는 3축 USP

  2. 식자재 트리플 (키위·케일·바나나) — 비주얼 임팩트가 강해 콘텐츠 자산 ↑

🧪 [제품 3] ABC 세럼 (광채/브라이트닝)

5월 말 출시 예정
현재 컨셉: 비타민A·B·C, NAD+, 식자재 기반. 광채/브라이트닝, 매일 사용 가능한 저자극. 직접 경쟁자: 코스알엑스 비타민C 23(리뷰 19,000건+), 구달 청귤 비타C E TXA, 멜라노CC, 폴라초이스
오늘 페인포인트매칭분석
세럼 #1 (코스알엑스 23 아침 밀림)🟢ABC 세럼의 결정적 진입 포인트. NAD+ 라이트 처방으로 "고함량의 효과를 아침에도" 가능하면 시장 공백 정확히 메움
세럼 #2 (밤 한정 → 아침 OK)🟢키위케일바나나와 동일한 메시지 축으로 확장 가능
세럼 #3 (화장 전 트리플 케어)🟡"화장 전 광채 베이스" 포지션 검토
🔴 즉시 결정 필요 (출시 4~5주 전)
  1. "아침에 발라도 흔적 없는 광채 세럼" 메인 USP 확정 — 코스알엑스 23의 1점 워딩(주황 자국·밀림·자극)을 정면 공략
  2. NAD+ 안정화 처방 임상 확보 — 코스알엑스 23의 "23%"처럼 정량적 USP 키워드 발굴 필수
  3. 저자극 테스트 결과 가시화 — "코스알엑스보다 순한" 명시적 비교가 가능한 데이터
🟡 검토 권장
  1. NAD+ 교육 콘텐츠 시리즈 — INKEY List 모델로 "NAD+가 뭔지 아세요?" 30초 시리즈
  2. 2주 톤 변화 콘텐츠 사례: 인플루언서 2주 사용 후 비교 컷 시딩
🟢 강점 (이미 잘 됨)

  2. 식자재 + 비건 — 코스알엑스(공산품 톤)와 톤 차별화 명확

🧼 [제품 4] 폼클렌징 (출시 준비 중)

출시 시점 미정
현재 컨셉: 흰목이버섯 다당류 (예상). 식자재 기반 약산성 클렌저. 직접 경쟁자: 라운드랩 1025 독도, 비플레인 녹두, 에스네이처 아쿠아 라이스, 해피바스 마이크로미셀라

폼클 #2 (1개로 피부 변화)🟡출시 후 첫 주 무료 샘플링/체험단으로 "3일 만에 거울이 달라지는" 사례 콘텐츠 확보 필수
폼클 #3 (아침 클렌징 젤 트렌드)🟡"하나로 아침저녁" 포지션 가능성 검토. 통합 클렌저 메시지 연구
🔴 즉시 결정 필요 (출시 6주 전)

  2. "흰목이버섯 다당류"를 마케팅 단일 키워드로 확정 — 듀이트리(아미노산), 에스네이처(쌀), 비플레인(녹두)처럼 단일 키워드 = 인지 효율
  3. 폼클렌징 단독 출시 vs 라인업 동시 출시 결정 — 사례 1(폼클 1개로 피부 변화)을 활용하려면 단독 출시 + 강력 시딩 전략이 유리
🟡 검토 권장
  1. "식자재 클렌저 신 카테고리" 포지셔닝: 쌀·녹두 다음으로 "버섯" 진입. 에스네이처 대비 "쌀뜨물 말고 버섯물" 같은 대비 카피
  2. 아침/저녁 통합 메시지: 클렌징 젤 트렌드와 직접 경쟁 가능한 텍스처 연구
🟢 강점 (이미 잘 됨)

  2. 식자재 라인업 통합 — 다른 카테고리(바디크림·세럼)와의 USP 일관성
