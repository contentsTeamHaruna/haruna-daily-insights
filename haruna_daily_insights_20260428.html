페인포인트 #1🆕 NEW

"끈적임 혐오로 바디로션 자체를 포기하는 소비자층 존재"

  • "끈적이는거 싫어해서 바디로션도 안바르는데 계속 건조"
  • "세타필 일리윤 건성용 같이 쓰거든 오늘부터 습해서 너무 끈적거리고 미치겠음"
  • "과도한 유분감의 고보습 바디로션을 바르면 끈적임과 텁텁함"
📍 직접 확인 가능 리뷰
  • 끈적이는거 싫어해서 바디로션도 안바르는데 (lostmindlab) — 끈적임 혐오 → 사용 포기 → 건조 악순환 직접 인용 출처
  • 여름에도 바디로션 꼭 쓰는 덬들 뭐 써? (더쿠) — 환절기 끈적임 급상승 불만 출처
  • 끈적임 없는 고보습 바디로션 추천 (언파) — 고보습=끈적임 딜레마 인용 출처
시장이 푸는 방식: 피지오겔 DMT(무향·산뜻), 일리윤 세라마이드(가벼운 제형), 액티브 야누스 풀 바디토닉(산뜻한 대안)
해결되지 않은 갭: "끈적임 0인데 보습도 충분한" 제품은 소비자 인식 속에 거의 없음. 끈적임이 싫어서 아예 바르지 않는 '비사용층'까지 존재 — 이 층은 시장 밖에 있어 경쟁이 없는 블루오션
하루나 진입 포인트 흰목이버섯 다당류의 가벼운 보습막 = "바디로션을 포기한 사람도 바를 수 있는 텍스처". "끈적임이 싫어서 안 바르셨죠? 이건 다릅니다" 카피로 비사용층 직접 공략

  • "발라도 건조해서 자꾸 다시 바르게 됨"
  • "흡수 쫩쫩 잘됨. 미끈거리는거 안남음 손에도 안남음 행.복." (만족 대비군)
📍 직접 확인 가능 리뷰
  • 건성피부 바디로션 추천좀 (디시 화장품갤) — "새벽에 가려워서 긁다 깸" 야간 건조 직접 인용 출처
  • 소소한 겨울 바디로션 리뷰 더마비/일리윤 (더쿠) — "발라도 건조해서 다시 바르게 됨" + 일리윤 만족 워딩 비교 출처
시장이 푸는 방식: 세타필 모이스춰라이징(야간용 추천 多), 일리윤 아토 세라마이드(보습 지속력 우수 평가), 닥터케라 우레아크림(각질+보습)
해결되지 않은 갭: 8시간 보습 지속을 '임상 데이터'로 증명한 바디크림이 거의 없음. 대부분 "느낌상 오래 가는 것 같다" 수준
하루나 진입 포인트 PDRN + 흰목이버섯 다당류 조합으로 "8시간 보습 지속 임상 테스트 완료" 같은 정량적 USP 제시 가능 시 결정적 차별점

📍 직접 확인 가능 리뷰
  • 모공각화증 피부 바디로션 추천 (언파) — "울글불긋 우둘투둘 스트레스" 직접 인용 출처
  • 닭살피부 인생 n년차 매끈해진 루틴 (디시) — 우레아 크림+바디로션 조합으로 개선 성공 사례
시장이 푸는 방식: 닥터케라 우레아크림 9%(각질 용해), 일리윤 세라마이드(꾸준히 바르면 개선), CeraVe SA 크림(살리실산)
해결되지 않은 갭: 우레아/SA 제품은 각질 제거력은 좋지만 "자극적"이라는 하위 페인이 있음. 순한 식자재 기반으로 모공각화증까지 케어하는 제품은 공백
하루나 진입 포인트 화이트 토마토 + 흰목이버섯 조합의 "순한 각질 케어 + 보습" 이중 효과. "닭살 고민인데 자극이 무서운 분" 타겟 콘텐츠 가능
🧪 세럼 — Top 3 페인포인트
페인포인트 #1🆕 NEW

"모공 세럼의 자극감 — 효과는 보이는데 매일 쓰기엔 따가움"

  • "화끈거림 & 따가움이 일시적이지만, 누군가에겐 정말 자극 요소"
  • "팩을 얹자마자 상큼한 향이 진하게 나고 5분도 안 돼서 따갑고 자극이 느껴짐"
  • "발효 성분이 안 맞으신다면 추천드리고 싶지 않은"
📍 직접 확인 가능 리뷰
  • 넘버즈인 3번 보들보들 결 세럼 후기 (언파) — "화끈거림, 따갑다" 자극 워딩 출처
  • 넘버즈인 3번 결세럼팩 파워리뷰 (언파) — "5분 만에 따갑고 자극" 발효 성분 불적합 인용 출처
시장이 푸는 방식: 넘버즈인 3번(팅글포어콤플렉스, 일시적 화끈거림 감수 → 결 개선), 메디힐 마데카소사이드(진정 중심, 자극 최소화)
해결되지 않은 갭: "모공 효과는 확실하면서도 매일 사용해도 자극 없는" 세럼이 시장에 부족. 효과적인 것은 자극적이고, 순한 것은 효과가 약한 구조
하루나 진입 포인트 키위케일바나나 세럼의 EGCG(녹차)+LHA/PHA 조합이 넘버즈인 대비 "순한 모공 케어" 포지셔닝 가능. "매일 써도 안 따가운 모공 세럼"
페인포인트 #2🆕 NEW

"브라이트닝 세럼 = 고함량이면 자극, 저함량이면 효과 없음"

  • "잡티에 효과" + "피부가 훨씬 광택있어보여" (구달 청귤 만족)
  • "살짝 점도있는 느낌이고 흡수는 빨리되는 편" (텍스처 만족)
  • 코스알엑스 비타C 23% — 즉각 톤업이나 매일 사용 불가 수준의 고함량 자극 (커뮤니티 반복 언급)
📍 직접 확인 가능 리뷰
  • 구달 청귤 비타C E TXA 세럼 체험단 후기 (더쿠) — "점도있지만 흡수 빠름", "광택 남아 피부 좋아보임" 인용 출처
  • 미백 비타민 중독자의 구달 청귤 비타C 후기 (더쿠) — "붉은 잡티 눈에 띄게 사라짐" 효과 인용 출처
시장이 푸는 방식: 구달 청귤(TXA 5%+나이아신아마이드 5%, 중함량 균형), 코스알엑스 비타C 23%(고함량 즉효), 원씽 나이아신아마이드(저자극 기본기)
해결되지 않은 갭: "순한데 2주 만에 눈에 보이는 브라이트닝"이라는 포지션이 비어 있음. 구달은 TXA 중심이라 비타C 순수주의자에겐 아쉽고, 코스알엑스는 너무 자극적
하루나 진입 포인트 ABC 세럼의 NAD+ + 식자재 기반 브라이트닝이 "매일 쓸 수 있는 저자극 광채 세럼" 포지션으로 구달과 코스알엑스 사이의 갭을 공략

  • 구달 청귤 비타C 세럼 후기 (더쿠) — "화장이 뜨거나 안함" 레이어링 호환성 워딩 출처
  • 넘버즈인 3번 결 세럼 내돈내산 (언파) — "끈적임 없이 산뜻" 텍스처 만족 출처
해석: 세럼의 1차 합격선은 여전히 흡수력·레이어링 호환성. 이번 주 습도 상승(60%+)과 함께 "산뜻한 흡수"에 대한 민감도가 더 올라가는 시즌
하루나 시사 키위케일바나나·ABC 모두 출시 전 "화장 위에 발라도 밀리지 않음" 텍스처 테스트 필수. 습한 봄~여름 시즌에 출시되므로 산뜻한 마무리감이 더욱 중요

  • 디시 클렌징폼 추천 약산성 세정력 (디시 화장품갤) — "약산성 쓰면 여드름, 세정력 강하면 싫다" 딜레마 직접 출처

해결되지 않은 갭: "약산성이면서 1회 세안으로 끝나는" 제품이 여전히 공백. 약산성 진영은 "2차 세안 필요", 약알칼리 진영은 "자극 우려"
하루나 진입 포인트 흰목이버섯 다당류 + 보조 식자재로 "약산성인데 한 번으로 끝나는" 포지션. 비플레인 녹두의 세정력 갭 + 라운드랩 독도의 1차 세안 한계를 동시 공략

  • SRB 쌀뜨물 효소세안제 후기 (언파) — "촉촉한 보습 클렌징" 식자재 기반 성공 사례
시장 분석: 쌀(에스네이처, SRB), 녹두(비플레인)가 식자재 클렌저 양대 진영. 소비자는 식자재 원료 = 순함·자연스러움으로 강하게 연상. 하지만 세정력 부족이라는 공통 한계

🧪 [제품 1] 키위케일바나나 세럼 (모공/진정)

5월 말 출시
메인 성분: EGCG(녹차) + 키위·케일 콤플렉스 + LHA/PHA + HA 8~11종 + NAD+
포지션: 모공/진정 → 지성·복합성·트러블 피부
직접 경쟁자: 넘버즈인 3번 보들보들 결 세럼 (올영 25,000건+ 리뷰)
오늘 페인포인트매칭분석
세럼 #1 "모공 세럼 자극감"🟢넘버즈인 3번의 최대 약점이 자극감. 키위케일바나나가 "순한 모공 세럼"을 증명하면 결정적 차별점. EGCG+LHA/PHA 조합이 팅글포어콤플렉스 대비 자극 낮을 수 있음
세럼 #3 "화장 밀림 흡수력"🟡HA 8~11종+다중 성분 조합이 무거워질 위험은 여전. 출시 전 습도 60%+ 환경 텍스처 테스트 필수
바디크림 #3 "모공각화증"(간접 🟢)모공 케어 세럼 사용자 중 바디 모공각화증도 함께 고민하는 층이 많음. "페이스+바디 모공 세트" 마케팅 가능성
🔴 즉시 결정 필요
  • 자극감 임상/체험 데이터 확보: 넘버즈인 3번 대비 "순한 모공 세럼" 포지셔닝의 핵심 근거. 최소 20명 이상의 민감피부 체험단 결과 필요
🟡 검토 권장
  • 출시 시즌(5월 말)과 텍스처: 습도 상승기에 산뜻한 마무리감이 더욱 중요. 고습도 환경 텍스처 최종 점검
  • "3주 사용 변화" 비주얼 자산: 출시 동시 릴스용 비포/애프터 이미지 사전 촬영 필요
🟢 강점
  • "순한 모공 세럼"이라는 포지션이 시장에 비어 있음: 넘버즈인이 자극, 메디힐이 진정 중심이라 "효과+순함" 동시 포지션은 명확한 갭
  • 피부타입 타겟이 명확: 지성·복합성·트러블 → 패키지·랜딩 첫 줄 명시 가능

🧪 [제품 2] ABC 세럼 (광채/브라이트닝)

5월 말 출시
메인 성분: NAD+ + 비타민 복합 + 식자재 기반
포지션: 광채/브라이트닝 → 건성·복합성 피부
직접 경쟁자: 구달 청귤 비타C TXA, 코스알엑스 비타C 23%, 원씽 나이아신아마이드
오늘 페인포인트매칭분석
세럼 #2 "브라이트닝 자극 vs 효과"🟢구달(중함량 밸런스)과 코스알엑스(고함량 자극) 사이에 "매일 쓸 수 있는 저자극 광채" 갭이 존재. NAD+가 이 갭을 채울 수 있음
세럼 #3 "화장 밀림"🟡구달 청귤의 5점 워딩이 "흡수 빠름+광택" → ABC도 이 수준의 흡수력 필수
🔴 즉시 결정 필요
  • "2주 만에 눈에 보이는 변화" 입증: 구달 청귤이 "2주 만에 잡티 옅어짐"으로 성공. ABC 세럼도 정량적 효과 데이터(톤 변화 수치, 멜라닌 감소 등) 필요
🟡 검토 권장
  • NAD+ 인지도 교육: 소비자 대부분이 NAD+를 모름. "NAD+가 뭔데요?"에 대한 1분 안에 이해되는 교육 콘텐츠 사전 제작 필요
  • 구달 청귤과의 가격 경쟁력: 구달이 가성비 포지션을 선점. 프리미엄으로 갈 것인지 가성비 경쟁할 것인지 결정
🟢 강점
  • "매일 쓸 수 있는 광채"라는 포지션이 명확: 코스알엑스(매일 사용 불가), 구달(TXA 중심)과 차별화된 NAD+ 스토리
  • 식자재 기반 + 광채의 조합이 시장에 신선: 브라이트닝 세럼 중 "식자재 원료" 컨셉은 거의 없음

🧼 [제품 3] 폼클렌징 (출시 준비 중)

출시 준비 중

직접 경쟁자: 비플레인 녹두, 에스네이처 쌀, 라운드랩 독도
오늘 페인포인트매칭분석
폼클 #1 "약산성=세정력 부족"🔴비플레인·라운드랩 모두 "2차 세안/1차 세안 필요"라는 동일 한계. 1회 세안 완결이 핵심 차별점이 되려면 세정력 포뮬러 검증 필수
폼클 #2 "식자재 신뢰+한계"🟢쌀·녹두 양대 진영에 "버섯"이라는 새 카테고리 진입 → 신선함 확보
폼클 #3 "거품 질감"🟡소비자가 "쫀쫀한, 몽글몽글한, 크리미한" 거품을 최우선시. 거품 질감 개발 테스트 결과 확인 필요
🔴 즉시 결정 필요
  • "1회 세안 완결" vs "데일리 순한 클렌저" 포지션 결정: 비플레인·라운드랩의 갭을 직접 공략하려면 세정력 강화 필수이나, 세정력을 올리면 순함이 희생될 수 있음. 어느 쪽에 무게를 둘 것인지 결정
  • 보조 식자재 결정: 녹두? 쌀? 다시마? 기존 식자재와 겹치지 않는 조합이 차별화의 핵심
🟡 검토 권장
  • 거품 질감 테스트: "쫀쫀하고 크리미한 거품"이 사용 만족도 1순위. 흰목이버섯 다당류가 거품 질감에 어떤 영향을 미치는지 확인
  • 가격 포지셔닝: 비플레인 녹두 16,900원/200ml, 에스네이처 쌀 12,000원/150ml, 라운드랩 독도 12,000원/200ml 대비 합리적 책정 필요
🟢 강점
  • "버섯" 식자재 클렌저라는 카테고리 자체가 시장에 없음: 쌀/녹두 양대 진영에 새 축 형성 가능
  • 흰목이버섯 다당류의 보습 + 세정 이중 기능: 글로벌 tremella 시장 38억달러(2033년) 성장 전망, "차세대 식물성 히알루론산" 포지셔닝 활용 가능
