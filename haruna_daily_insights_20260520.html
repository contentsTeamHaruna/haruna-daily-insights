🗣️ Part 1: Voice of Customer

🧴 바디크림/바디로션 — Top 3 페인포인트

#1 NEW “비 오는 날 바디로션 바르면 피부에 물기 섞여서 더 끈적해”

습도 70% 이상인 비 오는 날에는 바디로션의 유분이 피부 수분과 뒤섞여 끈적임 심화. 더쿠에서 계절 전환기 텍스처 불만 반복.
“겨울에 쓰다 남은거 쓰려니 흡수가 너무 느려서 바디로션 바르다 다시 땀남” “안바르면 건조하고 겨울에쓰던거 바르기엔 너무 무거워, 딱 중간인 게 없어” “여름되니까 바디로션 바르고 옷 입으면 옷에 다 묻어서 짜증나 그냥 안 바르게 돼”
📍 직접 확인 가능 리뷰
여름에 바를 산뜻한 바디로션 — 더쿠 여름에 쓰기 좋은 바디로션 추천 — 더쿠 바디로션은 항상 끈적이고 답답해서 잘 못 바르던 저 — 인스타그램
흰목이버섯 추출물 기반 겔 텍스처를 '습한 날에도 3초 흡수' 메시지로 포지셔닝. 비 오는 날 바디케어 루틴 콘텐츠로 고습도 사용감 차별화.

#2 지속 “바디로션 향이 오래 안 가, 비싼 거 안 쓰고 향 지속되는 제품 없나”

더쿠에서 "향 좋은 바디로션 추천" 류 글 꾸준. 인스타그램 #향좋은바디크림 해시태그와 함께 향수 레이어링용 바디크림 콘텐츠 트렌드.
“향 지속력 좋은 바디로션 없나, 바르고 1시간이면 향이 다 날아가” “조말론 바디크림은 향이 좋은데 너무 비싸, 가성비 있으면서 향 오래가는 거 추천” “바디로션 향이랑 향수 향이 섞이면 이상해서 무향 아니면 같은 계열 라인업이 필요함”
📍 직접 확인 가능 리뷰
바디로션 덕후 15년 후기 — 더쿠 오프온 바디로션 후기 — 더쿠 니치 향수 뷰티 트렌드 부상 — 뉴시스
식자재 기반 원료(바닐라, 코코넛, 시트러스)의 자연 향을 활용한 '향기 지속 바디크림' 포지셔닝. 향수 레이어링 가능한 은은한 자연 향조 개발.

#3 지속 “반팔 시즌인데 팔뚝 닭살(모공각화증) 관리법을 모르겠어”

초여름 습도 높아지는 시기에도 모공각화증 관련 검색 지속. 약사공론에서 "과도한 스크럽은 오히려 악화" 전문가 조언 게재.
“여름만 되면 민소매 입을 때 팔뚝 닭살 때문에 스트레스 엄청 받아” “모공각화증이라 각질제거+보습 같이 되는 바디크림 찾는데 제대로 된 게 없음” “스크럽 세게 하면 오히려 더 심해진다는데, 그냥 바르기만 하면 되는 바디크림은 없나”
📍 직접 확인 가능 리뷰
모공각화증 관리 — 약사공론 모공각화증 완화된 에디터의 바디로션 추천 — 원데이원 “닭살 때문에 반바지 못 입어요”...모공각화증이란? — 하이닥
화이트 토마토 추출물의 피부결 개선 효과를 '바르기만 하면 되는 닭살 케어'로 포지셔닝. "스크럽 없이 매일 보습으로 관리" 콘텐츠.
🧪 세럼 — Top 3 페인포인트

#1 NEW “PDRN이 진짜 효과 있는 건지, 후기만 많고 내 피부엔 안 맞을 수도”

코메디닷컴 "후기는 많은데 내 피부엔?" 기사 확산 이후, 더쿠·디시인사이드에서 PDRN 효능 회의론 급증. 글로우픽 세럼 1위가 비건 펩타이드 제품으로 교체.
“PDRN 100% 원액이라는데 실제 유효 PPM은 안 알려줘, 다 좋다고만 하니까 더 의심” “후기는 많은데 내 피부엔 안 맞을 수도 있잖아, 한 달 써봤는데 변화 1도 없음” “PDRN이 시술에서 온 이름인데 바르는 걸로 같은 효과 날 리 없잖아”
📍 직접 확인 가능 리뷰
“후기는 많은데 내 피부엔?”…PDRN 화장품 — 코메디닷컴 PDRN 화장품 ‘이것’과 함께 쓰면 부작용 생길 수도 — 코메디닷컴 글로우픽 에센스/세럼 카테고리 TOP20 — 글로우픽
키위케일바나나 세럼을 'PDRN 피로감 속 식물성 대안'으로 포지셔닝. "성분 함량 100% 투명 공개, 식자재 원료로 만든 비건 재생 세럼" 메시지.

#2 지속 “세럼 종류가 너무 많아서 뭘 써야 할지 모르겠어”

더쿠에서 "제품이 넘 많아서 뭘 써야할지 모르겠어" 반응 반복. 디시인사이드 미니멀리스트 진영에서 "세럼 불필요론"까지 등장하며 소비자 의사결정 피로감 심화.
“제품이 넘 많아서 뭘 써야할지 모르겠어, PDRN인지 펩타이드인지 비타민C인지” “세럼 토너 에센스 앰플 다 뭐가 다른 건데, 하나만 쓰면 안 되나” “결국 클렌저+보습+선크림이면 충분하다는 의견도 있던데 세럼이 진짜 필요한 거야?”
📍 직접 확인 가능 리뷰
비타민 C 세럼 뭐써 — 더쿠 브링그린 징크테카 세럼 2개월 후기 — 더쿠 아침에 바를만한 세럼 추천 좀 — 디시인사이드
"이것 하나면 끝" 올인원 포지셔닝. 키위(비타민C)+케일(항산화)+바나나(보습) 3가지 식자재의 역할을 명확히 설명하여 "왜 이 세럼 하나로 충분한지" 교육 콘텐츠.

#3 지속 “세럼 바르면 끈적이고 선크림이랑 밀려, 아침엔 생략하게 돼”

비 오는 습한 날에 더 심해지는 끈적임 문제. 지성 피부 중심 "아침 루틴에서 세럼 빼자" 의견 확산. 디시인사이드에서 추천 요청 지속.
“세럼 바르고 선크림 바르면 지우개똥처럼 밀려서 여름엔 세럼 생략하게 됨” “PDRN 앰플 좋은 건 알겠는데 끈적여서 아침에는 못 쓰겠어” “습한 날엔 세럼까지 바르면 얼굴이 번들거려서 차라리 토너만 바름”
📍 직접 확인 가능 리뷰
세럼 추천좀 해줘 — 디시인사이드 PDRN 앰플 추천 TOP3 솔직 비교 — 언니의파우치 구달 청귤세럼 트러블 — 더쿠
ABC 세럼을 '워터 타입 초경량 제형'으로 설계. "비 오는 습한 날에도 밀림 없는 세럼" 콘텐츠로 아침 루틴 필수템 포지셔닝.
🧼 폼클렌징 — Top 3 페인포인트

#1 NEW “이중세안 귀찮은데, 폼 하나로 선크림까지 깔끔히 지울 수 있는 게 없나”

더페이스샵 밀크앤젤리 더블클렌징폼 글로우픽 1위 차지하며 '이중세안 원스텝' 수요 가시화. 디시인사이드에서 "무기자차도 2중세안 필수" 공감글 화제.
“클렌징오일 쓰고 폼까지 쓰는 게 너무 귀찮아, 하나로 다 되는 거 없나” “무기자차 써도 2중세안은 똑같구나, 선크림 클렌징이 제일 귀찮은 단계” “팩클렌저니 밀크폼이니 종류가 너무 많아져서 뭘 써야 할지 모르겠음”
📍 직접 확인 가능 리뷰
글로우픽 클렌징폼 카테고리 TOP20 — 글로우픽 폼클렌징 30개 써보고 추천 — 디시인사이드 내가 써본 클렌징폼 13종 모음 — 디시인사이드
하루나 폼클렌징을 '원스텝 딥클렌저'로 포지셔닝. 코코넛 유래 계면활성제로 선크림까지 단번에 제거. "이중세안 없는 날" 콘텐츠.

#2 지속 “약산성 클렌징폼 쓰다가 좁쌀여드름 올라왔어”

더쿠에서 약산성 역효과 경험담 공감. 디시인사이드 "아침 약산성, 저녁 약알칼리" 절충안 합의. 토리든 pH 7 논란(이후 5.8 개선)도 신뢰도 하락 요인.
“약산성이라길래 샀는데 며칠 내로 피부가 가렵고 오돌토돌한 게 올라왔어” “약산성 vs 약알칼리 뭐가 맞는지 모르겠어, 약산성 쓰니까 기름기가 안 빠지는 느낌” “토리든이 약산성이라 광고했는데 pH 측정하니 7이었다는 게 충격”
📍 직접 확인 가능 리뷰
약산성 쓰다가 좁쌀여드름 — 더쿠 약산성 안맞는 사람용 알칼리성 모음 — 더쿠 약산성 약알칼리 종결내준다 (정성글) — 디시인사이드
'진짜 약산성 pH 5.5' — 독립 기관 pH 인증 결과 패키지 명기. "약산성이라고 다 같지 않다, pH 실측치 공개" 투명성 전략.

#3 지속 “세안 후 당김이 심해서 바로 스킨케어 안 하면 얼굴이 갈라질 것 같아”

비 오는 습한 날에도 세안 후 당김 호소 지속. 언니의파우치 "건성 클렌징폼" 질문 꾸준. 아미노산 계면활성제 기반 폼 대안 부상.
“세안하고 나면 얼굴이 바로 당기고 하얗게 일어나는 느낌, 볼 쪽 각질도 들뜸” “폼클렌징 거품 많이 내면 개운한 줄 알았는데 그게 피부장벽 깎는 거였어” “기초 발라도 흡수가 잘 안 되는 건 세안제 때문이라는 걸 이제야 알았음”
📍 직접 확인 가능 리뷰
건성 클렌징폼 뭐가 좋을까요… 너무 건조해요 — 언니의파우치 아침 세안만 바꿨을 뿐인데 — W Korea 봄~여름 지성 추천 LIST — 디시인사이드
아미노산 계면활성제 + 코코넛·녹두 식자재 보습으로 '세안 후 당김 제로'. "씻는 순간부터 보습이 시작되는 폼" 메시지.
