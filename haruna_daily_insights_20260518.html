🗣️ Part 1: Voice of Customer

🧴 바디크림/바디로션 — Top 3 페인포인트

#1 NEW “초여름에 바디크림 바르면 옷에 다 묻어, 흡수가 안 돼”

5월 중순 기온 28도를 넘기면서 겨울철 고보습 바디크림을 그대로 사용하는 소비자들의 끈적임 불만 폭발.
“여름되니까 바디로션 바르고 옷 입으면 옷에 다 묻어서 짜증나 그냥 안 바르게 돼” “아침에 바디크림 바르고 출근하면 팔꿈치에 하얗게 뭉쳐있어서 민망해” “보습력은 좋은데 흡수가 너무 느려서 여름엔 못 쓰겠음, 가벼운 거 추천 좀”
📍 직접 확인 가능 리뷰
여름에 바를 산뜻한 바디로션 머없을까? — 더쿠 보습력 좋고 안끈적이는 바디로션 뭐있어? — 더쿠 바디로션 브랜드 10종 비교 — 한국소비자원
실키 머쉬룸 바디크림의 가벼운 겔 텍스처를 '3초 흡수, 옷에 안 묻는 바디크림'으로 포지셔닝. 출근 전 루틴 콘텐츠로 시간 절약 메시지 전달.

#2 지속 “팔뚝 닭살(모공각화증) 여름이라 더 신경 쓰여”

기온 상승과 반팔 착용 시작으로 모공각화증 관련 검색량 급증. 노출 부위 관리 수요 최고점.
“여름만 되면 민소매 입을 때 팔뚝 닭살 때문에 스트레스 엄청 받아” “모공각화증이라 각질제거+보습 같이 되는 바디크림 찾는데 제대로 된 게 없음” “닭살 때문에 반팔도 못 입어, 바디스크럽 하고 로션 발라도 그대로야”
📍 직접 확인 가능 리뷰
모공각화증 완화된 에디터의 바디로션 추천 — 원데이원 모공각화증에 좋은 바디제품 추천 — 유로라이프 디어도어 비건 바디스크럽 — 올리브영
화이트 토마토 추출물의 피부결 개선 효과를 모공각화증 관리 앵글로 연결. "반팔 시즌 팔뚝 케어" 콘텐츠.

#3 NEW “바디케어도 '투 스텝'으로 해야 한다는데, 뭘 써야 하지?”

2026년 바디케어 트렌드가 역할 분담 체계로 변화. 소비자들은 새 루틴에 대한 정보 과부하로 혼란.
“바디케어도 스킨케어처럼 단계별로 해야 한다는데 뭐부터 사야 할지 모르겠어” “등드름은 미스트, 보습은 로션이라는데 둘 다 바르면 끈적이지 않나?” “바디세럼이라는 것도 나오던데 로션이랑 뭐가 다른 건지 개념이 안 잡혀”
📍 직접 확인 가능 리뷰
화해 월간 랭킹 2026년 2월 바디케어 트렌드 — 화해 여름에도 쓰기 좋은 바디로션 추천해줄래? — 더쿠 에디터가 고른 여름 뷰티템 5 — 하퍼스 바자
하루나 바디크림을 '올인원 바디 솔루션'으로 포지셔닝. 흰목이버섯(보습) + 화이트토마토(진정)의 멀티 기능으로 "이거 하나면 충분" 메시지.
🧪 세럼 — Top 3 페인포인트

#1 NEW “PDRN 세럼 후기는 넘쳐나는데 내 피부에 맞을지 불안해”

PDRN이 기본 스펙으로 안착했으나 제품별 부원료 다양화로 선택 불안 확산. 어류 유래 불순물 우려도 제기.
“PDRN 앰플 사고 싶은데 어류 추출이라 비린내 나진 않을지, 알레르기 반응 없을지 걱정” “요즘 PDRN 제품 너무 많아서 뭘 골라야 할지 모르겠음, 다 좋다고만 하니까 더 헷갈려” “PDRN이랑 레티놀 같이 썼다가 피부 빨개져서 고생했어, 성분 조합 주의해야 함”
📍 직접 확인 가능 리뷰
“후기는 많은데 내 피부엔?” PDRN 화장품 — 코메디닷컴 PDRN 화장품 '이것'과 함께 쓰면 부작용 — 코메디닷컴 다이소 PDRN 라인 꼭 써봐 미쳤음 — 더쿠
키위케일바나나 세럼의 식자재 기반 원료로 어류 유래 PDRN 불안감 해소. '식물성 재생 세럼' 포지셔닝으로 차별화.

#2 지속 “세럼 바르면 끈적이고 다음 단계 밀림 현상 심해”

고기능 세럼의 점도 문제. 특히 선크림과 레이어링 시 밀림 현상이 여름철 주요 이탈 원인.
“세럼 바르고 선크림 바르면 지우개똥처럼 밀려서 여름엔 세럼 생략하게 됨” “PDRN 앰플 좋은 건 알겠는데 끈적여서 아침에는 못 쓰겠어” “여름에는 세럼 대신 에센스 토너로 대체하는 사람 나만 있나”
📍 직접 확인 가능 리뷰
PDRN부터 엑소좀까지! 세럼 TOP 7 — 하퍼스 바자 메디큐브 PDRN 핑크 콜라겐 엑소좀 샷 — 언니의파우치 VT 리들샷 & PDRN 에센스 후기 — 더쿠
ABC 세럼 개발 시 '워터 타입 제형'으로 설계. "선크림 전에 바르는 초경량 세럼" 포지셔닝.

#3 NEW “비싼 세럼 효과 체감 어려워, 가성비 PDRN 찾는 중”

다이소 5,000원 PDRN이 화제. 가격 양극화 속 '합리적 효능' 기준점 형성 중.
“다이소 PDRN이 5천원인데 3만원짜리랑 뭐가 다른 건지 솔직히 모르겠음” “비싼 세럼 한 달 쓰고 피부 변화 1도 없어서 돈 아깝다는 생각만 듦” “성분 함량이 중요하다는데 어차피 표기도 안 해주니까 비교가 안 됨”
📍 직접 확인 가능 리뷰
다이소 pdrn 상품 추천해주라 — 더쿠 20만개 완판 3세대 PDRN 엑소좀 앰플 — 메이비나 PDRN 세럼, 잘 고르면 시술 효과? — 얼루어 코리아

🧼 폼클렌징 — Top 3 페인포인트

#1 NEW “세안 후 당김이 심해서 바로 스킨케어 안 하면 갈라질 것 같아”

약알칼리성 폼클렌징의 과도한 세정력. 여름철 자외선+냉방으로 피부장벽 약화 상태에서 증상 심화.
“세안하고 나면 얼굴이 바로 당기고 하얗게 일어나는 느낌, 볼 쪽 각질도 들뜸” “폼클렌징 거품 많이 내면 개운한 줄 알았는데 그게 피부장벽 깎는 거였어” “기초 발라도 흡수가 잘 안 되는 건 세안제 때문이라는 걸 이제야 알았음”
📍 직접 확인 가능 리뷰
건성 클렌징폼 뭐가 좋을까요… 너무 건조해요 — 언니의파우치 속당김 없는 폼클렌징 리뷰 — 뜨엘 아침 세안만 바꿨을 뿐인데 — W Korea
하루나 폼클렌징을 '세안 후 당김 제로' 컨셉으로 개발. 식자재 기반 계면활성제로 피부장벽 보호 세정.

#2 지속 “약산성이라고 다 순한 건 아니더라, pH만 낮추고 자극 성분 그대로”

약산성 마케팅 보편화 속 소비자 눈높이 상승. pH만 맞추고 SLS/합성향 그대로인 제품에 불신 확산.
“약산성이라길래 샀는데 쓰고 나면 뻑뻑한 느낌, pH만 약산성이고 성분은 자극적인 듯” “약산성 vs 약알칼리 뭐가 더 좋은지 아직도 논쟁 중인데 결국 성분 조합이 중요한 거 아님?” “토리든 약산성은 마무리감 좋은데 다른 약산성 제품은 오히려 더 당겨”
📍 직접 확인 가능 리뷰
약산성 약알칼리 종결내준다 — 디시인사이드 폼클렌징 30개 써보고 추천 — 디시인사이드 약산성 폼클렌징 왜쓰는거임? — 디시인사이드
'진짜 순한 약산성' — pH뿐 아니라 전 성분 EWG 그린 등급 기준 충족. 식자재 원료 기반 자연 유래 계면활성제.

#3 NEW “여름에 선크림 쓰는데 폼으로 부족해, 이중세안 귀찮아”

자외선 차단제 사용 증가로 폼 단독 세정 불완전. 이중세안 번거로움 불만 동시 존재.
“선크림 워터프루프 쓰면 폼으로만 안 지워져서 결국 오일 먼저 써야 하는데 귀찮음” “이중세안 하면 피부 더 건조해지는 것 같은데 안 하면 모공에 선크림 잔여물 남을 것 같고” “선크림까지 한 번에 깔끔히 지워지면서 순한 클렌저 있으면 좋겠다”
📍 직접 확인 가능 리뷰
순한 클렌징폼 추천 피부타입별 가이드 — 봉지아 피부장벽 지킴이 약산성 클렌징폼 기획전 — 올리브영 메이크프렘 PDRN 클렌징 밀크 후기 — 더쿠
'원스텝 딥클렌징' — 식자재 유래 마일드 오일 성분 탑재로 선크림까지 한 번에 제거. "이중세안 필요 없는 순한 딥클렌저".
