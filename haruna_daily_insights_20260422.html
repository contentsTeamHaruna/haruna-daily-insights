⚔️ 미샤 PDRN NAD+ 라인 vs 하루나 — 경쟁 분석

미샤가 대중 브랜드로서 PDRN + NAD+ 3스텝 라인을 출시

  • "NAD+" 키워드 선점 경쟁 시작 — 5월 말 하루나 출시 시 미샤가 먼저 노출될 가능성. "식자재 기반 NAD+" 차별화 내러티브 더욱 중요
  • 가격대 비교 필요 — 미샤 예상 2~3만원대. 프리미엄 세그먼트 차별화 or 가성비 경쟁력 확보 재검토
  • 3스텝 시스템 vs 단품 — 하루나도 바디크림+세럼+폼클렌징 "하루나 3스텝 루틴" 시스템 소구 기회
  • 역이용 전략 — 미샤의 시장 교육 효과(NAD+ 인지도 확산)를 활용, "식자재 기반 프리미엄"으로 차별 진입
🧴 기능성 바디케어 트렌드 — 하루나 바디크림 포지셔닝 강화

페이스케어 활성 성분의 바디 카테고리 확산 가속

  • 더마팩토리 나이아신아마이드 바디로션 — DCInside "바디에도 미백 성분?" 논의. 기능성 바디케어 수용도 상승
  • 오프온 리페어 바디로션 — 더쿠 핫글. 리페어·재생 기능 바디케어 수요 확인
  • 하루나 적용 — PDRN(재생) + 흰목이버섯(보습) + 화이트 토마토(브라이트닝) USP를 "기능성 바디케어의 정점"으로 재포지셔닝
